# 💀 TROLL RANSOMWARE SIMULATION 💀

⚠️ **THIS IS A HARMLESS TROLL SIMULATION - NOT REAL RANSOMWARE** ⚠️

A funny prank simulation that looks like ransomware but does absolutely NO harm to your computer. No files are encrypted, no data is stolen, nothing is damaged. It's just a visual joke!

## 🎭 What is this?

This is a Python script that creates a full-screen ransomware-like interface as a prank. When run, it shows:

- A red screen with "YOUR COMPUTER IS NOW IN THEIR HANDS"
- A 2-hour countdown (just for show)
- A password input field

## 🔑 The "Secret" Password

Enter `123456789` to "unlock" the computer and see the green success screen!

## 🛡️ Is it safe?

**100% SAFE!** This script:
- ✅ Does NOT encrypt any files
- ✅ Does NOT steal any data
- ✅ Does NOT damage your system
- ✅ Does NOT spread to other computers
- ✅ Does NOT require administrator privileges
- ✅ Does NOT modify registry or system files
- ✅ Can be closed by entering the password
- ✅ Can be killed from Task Manager if needed
